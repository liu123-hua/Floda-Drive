# 项目简介
    本项目是一个类似百度网盘的网盘项目。
# 技术选型
1. SSM+mysql+ftp
2. layUI
# 功能模块
### 用户模块
- 用户注册（邮箱验证码验证）
- 密码重置
- 基础信息管理
### 文件模块
- 文件上传
- 文件下载(多文件打包成.zip)
- 文件分类
- 常用格式的文件在线预览
- 文件重命名
- 文件删除
### 文件夹模块
- 新建文件夹
- 删除文件夹
- 重命名文件夹
### 分享模块
- 新建分享
- 通过链接和二维码访问分享
- 删除分享
### 回收站模块
- 删除回收站的文件
- 还原文件
- 清空回收站
